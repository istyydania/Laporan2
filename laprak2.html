RINGKASAN

HTML (HyperText Markup Language) digunakan untuk membangun struktur dan konten dasar dari sebuah halaman web. Dimana HTML ini menandai suatu struktur. Elemen utama dari HTML berupa:

  • Heading atau (<h1> sampai <h6>) fungsinya untuk menampilkan judul
  • Paragraf atau (<p>) fungsinya menampilkan teks
  • Gambar atau (<img>) fungsinya untuk menyisipkan sebuah gambar
  • Tautan atau (<a>) fungsinya untuk menambahkan hyperlink
Selain itu terdapat dokumen HTML yang memiliki struktur yang terdiri dari deklarasi doctype, tag <html>, <head>, <title>, dan <body>, yang menyusun konten utama halaman. Kemudian ada banyak framework CSS, seperti Bootstrap dan Tailwind CSS, yang menyediakan komponen dan gaya siap pakai untuk mempercepat pengembangan.

CSS (Cascading Style Sheets) adalah bahasa stylesheet yang digunakan untuk memperindah dan mengatur tata letak halaman web. Selain itu CSS ini dapat digunakan juga untuk mendesain atau mempercantik tampilan halaman web dengan fitur utama seperti:

  • Warna: color, background-color
  • Font: font-family, font-size
  • Tata Letak: margin, padding, border

JavaScript menyediakan action kemudian user bisa lebih interaktif terhadap website atau aplikasi yang dibuat, dan memungkinkan menganimasikan konten. Pada JavaScript ini memiliki pustaka yang dinamakan jQuery, yang dirancang untuk memungkinkan pengembang untuk menulis kode yang lebih sedikit dan lebih mudah dibaca untuk melakukan tugas-tugas dalam pengembangan web.

I. Mengapa menggunakan HTML, CSS, maupun JavaScript dalam pemrograman web?

Dikarenakan pada 3 komponen ini sangat penting dalam pengembangan sebuah web. HTML untuk memberikan struktur, CSS memberikan desain, dan JavaScript memberikan interaktivitasnya.

II. Bagaimana HTML, CSS, dan JavaScript dapat diintegrasikan dalam pemrograman web?

Dimana HTML sebagai suatu struktur dalam membangun kerangka dasar halaman web dengan elemen seperti judul, paragraf, dan tombol. Kemudian untuk CSS sebagai desain, CSS ini mengatur tampilan halaman, memberikan gaya pada elemen HTML seperti warna dan ukuran font, sehingga halaman terlihat menarik. Untuk JavaScript menambahkan interaktivitas, kemudian memungkinkan pengguna berinteraksi dengan elemen halaman, seperti mengklik tombol untuk mengubah teks. Ketiga komponen ini saling terhubung, dengan kombinasi ini, halaman web menjadi fungsional, menarik, dan interaktif.
